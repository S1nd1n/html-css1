Введение в списки

Списки могут содержать:
  • Неупорядоченную информацию (маркированный список - UL)
  • Упорядоченную информацию (нумерованный список - OL)
  • Определения (спискок определений - DL)

1. Маркированный список UL

Маркеры списка (list-style-type): Диск (disc) Кружок (circle) Квадрат (square) Любой символ ("- ") Символ Unicode ("\1440 ") Картинка

2. Нумерованный список OL

Варианты нумерации (list-style-type): Десятичная (decimal) Маленькими буквами (lower-alpha) Заглавными буквами (upper-alpha) Римскими маленькими (lower-roman) Римскими заглавными (upper-roman) Армянская (armenian) Грузинская (georgian) Катакана (katakana) Греческая (lower-greek)

3. Спискок определений DL

Список терминов HTML HyperText Markup Language - язык разметки CSS Cascading Style Sheets - таблицы стилей XML eXtensible Markup Language - расширяемый язык разметки XSL eXtensible Stylesheet Language - расширяемый язык описания стилей
